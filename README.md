# 🐱 Голодный кот — обучающая игра

## Описание
Интерактивная игра-тренажёр для проверки знаний в рамках курса по дисциплине "Веб-дизайн". Игрок ловит мышек, отвечая на вопросы. За правильные ответы начисляются очки, за ошибки — игра заканчивается.

## Функциональность
- 3 уровня сложности
- 15??? встроенных вопросов по HTML/CSS
- Звуковое сопровождение (Web Audio API)
- Рейтинг победителей (localStorage)
- Анимации мышек (поймана/убежала)

## Технологии
- HTML/CSS
- JavaScript 
- Web Audio API

## Запуск
1. Клонировать репозиторий
2. Открыть `index.html` в браузере
3. Либо использовать Live Server в VS Code
